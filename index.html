This web page showcases the images rendered by our custom raytracer. The raytracer computes light paths to generate images with realistic shadows and reflections. It supports geometry like spheres and triangles alongside complex model transformations.

Core Features & Optimizations

1. Uniform Grid Acceleration Structure

  • We implemented a Uniform Grid to partition the scene into voxels to reduce intersection tests by only checking objects within cells that the ray actually passes through.
  • Used the 3DDDA algorithm for fast grid stepping, ensuring optimal performance in scenes with high object counts, such as the Stanford Dragon.
  • Automatically calculates the optimal grid dimensions based on scene bounds and object density using a cube-root heuristic.
  • World-space AABB for all primitives were pre-computed to ensure efficient testing.

2. Supersampling Anti-Aliasing (SSAA)

We added support for anti-aliasing to smooth out jagged edges and improve overall image quality. This is Jittered Supersampling, which casts multiple rays per pixel with small random sub-pixel offsets. The results are averaged to produce the final pixel color, providing high-quality edge aliasing mitigation.

  • Allows configurable Samples Per Pixel (SPP) via scene files using a jittered grid approach to sample sub-pixel positions randomly.
  • Effectively removes jagged edges on geometric silhouettes and smoothes high-frequency artifacts in specular highlights and shadows.

3. Environment Mapping (HDRI)

Used Environment Mapping to simulate lighting bounds. By loading an HDRI image, rays that do not hit any objects will sample the environment map's pixel data based on their spherical coordinates mapping relative to the HDRI. Reflective objects in the scene accurately reflect the surrounding environment map.

  • Integrated FreeImage to load and sample HDRI maps (.hdr/.exr) to provide more realistic background lighting and reflections.
  • Implemented equirectangular lookups based on ray direction vectors for a 360-degree skybox.
  • Recursive rays that exit the scene sample the environment map; reflective surfaces mirror the HDRI environment more accurately.
